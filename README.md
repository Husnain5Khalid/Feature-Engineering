## 📌 Project Overview
This project focuses on performing feature engineering on a salary dataset to improve data quality and create meaningful features for machine learning.

---

## 📊 Dataset Description
The dataset contains the following columns:
- job_title  
- experience_years  
- education_level  
- skills_count  
- industry  
- company_size  
- location  
- remote_work  
- certifications  
- salary  

---

## ⚙️ Feature Engineering Steps
The following features were created:

- **experience level**  
- **education status**
- **skills level** 
- **skills_per_year** 
- **salary based on skills and experince** 

---

## 🛠️ Tools Used
- Python  
- Pandas  
- NumPy  
- Jupyter Notebook  

---



---

## 📈 Goal of the Project
To improve model performance by creating meaningful features from raw data and better understand salary patterns.
